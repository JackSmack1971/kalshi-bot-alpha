string

header

string

header

required

](https://docs.kalshi.com/api-reference/portfolio/get-subaccount-transfers#parameter-limit)

limit

integer<int64>

default:100

Number of results per page. Defaults to 100.

Required range: `1 <= x <= 1000`

[​

](https://docs.kalshi.com/api-reference/portfolio/get-subaccount-transfers#parameter-cursor)

cursor

string

Pagination cursor. Use the cursor value returned from the previous response to get the next page of results. Leave empty for the first page.
