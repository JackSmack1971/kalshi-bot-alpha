# [](https://spec.openapis.org/registry/format/unixtime.html#format-registry)[Format Registry](https://spec.openapis.org/registry/format/)

## [](https://spec.openapis.org/registry/format/unixtime.html#unixtime---seconds-since-jan-1st-1970---ieee10031-2024posix1-2024)unixtime - seconds since Jan 1st 1970 - [IEEE1003.1-2024/POSIX.1-2024](https://pubs.opengroup.org/onlinepubs/9799919799/)

JSON Data Type: `number, string`.

The `unixtime` format represents seconds since Jan 1st 1970 - defined as 4.19 Seconds Since the Epoch - [IEEE1003.1-2024/POSIX.1-2024](https://pubs.opengroup.org/onlinepubs/9799919799/).
