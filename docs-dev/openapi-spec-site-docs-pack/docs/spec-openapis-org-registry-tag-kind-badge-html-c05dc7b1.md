# [](https://spec.openapis.org/registry/tag-kind/badge.html#tag-kind-registry)[Tag Kind Registry](https://spec.openapis.org/registry/tag-kind/)

## [](https://spec.openapis.org/registry/tag-kind/badge.html#badge----tags-with-kind-badge-are-applied-as-visible-badges-in-documentation)badge - Tags with `kind: badge` are applied as visible badges in documentation.

Tags with `kind: badge` are applied as visible badges in documentation. It is expected that an operation might have many badges.
