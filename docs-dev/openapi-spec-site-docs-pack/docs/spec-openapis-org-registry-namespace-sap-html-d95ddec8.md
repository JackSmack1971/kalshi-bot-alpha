# [](https://spec.openapis.org/registry/namespace/sap.html#extension-field-namespace-registry)[Extension Field Namespace Registry](https://spec.openapis.org/registry/namespace/)

## [](https://spec.openapis.org/registry/namespace/sap.html#sap---extensions-created-and-used-by-sap)sap - Extensions created and used by SAP

The `x-sap-` prefix is reserved for extensions created by SAP. These extensions are available for use by anyone.

The official list of supported SAP extensions can be found in [SAP/openapi-specification](https://github.com/SAP/openapi-specification) repository.
