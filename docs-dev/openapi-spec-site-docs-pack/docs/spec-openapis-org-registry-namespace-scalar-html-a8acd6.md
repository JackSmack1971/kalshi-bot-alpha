# [](https://spec.openapis.org/registry/namespace/scalar.html#extension-field-namespace-registry)[Extension Field Namespace Registry](https://spec.openapis.org/registry/namespace/)

## [](https://spec.openapis.org/registry/namespace/scalar.html#scalar---extensions-created-and-used-by-scalar)scalar - Extensions created and used by Scalar

The `x-scalar-` prefix is reserved for extensions created by Scalar. These extensions are available for use by anyone.

The official list of supported Scalar extensions can be found in [scalar/scalar](https://guides.scalar.com/scalar/scalar-api-references/openapi#openapi-specification__custom-specification-extensions) repository.
