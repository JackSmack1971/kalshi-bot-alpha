# [](https://spec.openapis.org/#openapi-initiative-publications)OpenAPI Initiative Publications

This site contains the authoritative HTML renderings of the OpenAPI Initiative’s [specifications](https://spec.openapis.org/#specifications) and [extension registries](https://spec.openapis.org/#registries).

Please see the [Learn OpenAPI](https://learn.openapis.org/) site for additional documentation and [examples](https://learn.openapis.org/examples/), and the [OpenAPI Tooling](https://tools.openapis.org/) site for community-provided lists of tools implementing the specifications.

## [](https://spec.openapis.org/#specifications)Specifications

Each specification page lists all published specification versions and official (but non-[normative](https://en.wikipedia.org/wiki/Normativity#Standards_documents)) schema iterations.

-   [Arazzo Specification](https://spec.openapis.org/arazzo/)
-   [OpenAPI Specification](https://spec.openapis.org/oas/)
-   [Overlay Specification](https://spec.openapis.org/overlay/)

## [](https://spec.openapis.org/#registries)Registries

The [Registry Page](https://spec.openapis.org/registry/index.html) includes documentation as well as API and RSS access for all registries

Registry shortcuts:

-   [Extension Field Registry](https://spec.openapis.org/registry/extension)
-   [Extension Field Namespace Registry](https://spec.openapis.org/registry/namespace)
-   [Format Registry](https://spec.openapis.org/registry/format)
-   [Tag Kind Registry](https://spec.openapis.org/registry/tag-kind)
-   [Media Type Registry](https://spec.openapis.org/registry/media-type)
-   [Draft Features Registry](https://spec.openapis.org/registry/draft-feature)
-   [Alternative Schema Type Registry](https://spec.openapis.org/registry/alternative-schema)
