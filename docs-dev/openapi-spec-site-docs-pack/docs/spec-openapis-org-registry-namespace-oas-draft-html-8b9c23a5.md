# [](https://spec.openapis.org/registry/namespace/oas-draft.html#extension-field-namespace-registry)[Extension Field Namespace Registry](https://spec.openapis.org/registry/namespace/)

## [](https://spec.openapis.org/registry/namespace/oas-draft.html#oas-draft---extensions-created-by-oai-to-indicate-proposed-changes-to-the-oas-specification)oas-draft - Extensions created by OAI to indicate proposed changes to the OAS specification

The `x-oas-draft-` prefix is reserved for extensions created as part of the [draft features](https://github.com/OAI/OpenAPI-Specification/blob/main/DEVELOPMENT.md#draft-features) process. These extensions are available for use by anyone.
