## [](https://spec.openapis.org/registry/format/relative-json-pointer.html#relative-json-pointer---a-json-string-representation-of-a-relative-json-pointer-as-defined-in-draft-rfc-01)relative-json-pointer - A JSON string representation of a relative JSON Pointer as defined in draft RFC 01

JSON Data Type: `string`.

The `relative-json-pointer` format is a JSON string representation of a relative JSON Pointer as defined in draft RFC 01.

[Relative JSON pointers draft RFC 01](https://datatracker.ietf.org/doc/html/draft-handrews-relative-json-pointer-01).
