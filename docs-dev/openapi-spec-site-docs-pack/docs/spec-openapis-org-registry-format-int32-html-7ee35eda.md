## [](https://spec.openapis.org/registry/format/int32.html#int32---signed-32-bit-integer)int32 - signed 32-bit integer

JSON Data Type: `number`.

The `int32` format represents a signed 32-bit integer, with the range −2,147,483,648 through 2,147,483,647. This format entry is to ensure future versions of OpenAPI maintain compatibility with [OpenAPI 3.0.x](https://spec.openapis.org/oas/v3.0.0).
