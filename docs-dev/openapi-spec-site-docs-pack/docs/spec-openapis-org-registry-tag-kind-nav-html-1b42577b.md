# [](https://spec.openapis.org/registry/tag-kind/nav.html#tag-kind-registry)[Tag Kind Registry](https://spec.openapis.org/registry/tag-kind/)

## [](https://spec.openapis.org/registry/tag-kind/nav.html#nav---tags-with-kind-nav-are-used-in-documentation-to-group-operations-into-sections)nav - Tags with `kind: nav` are used in documentation to group operations into sections

Tags with `kind: nav` are used in documentation to group operations into sections. In most cases, an operation only has one tag of this type so it belongs in one section.
