MINUTES OF THE REGULAR MEETING OF THE TECHNICAL STEERING COMMITTEE

13 September 2018

A regular meeting of the Technical Steering Committee (the “TSC”) of the OpenAPI Initiative (“OAI”) was held on 13 September 2018 at 9:00 a.m. PT via conference call.

Consideration moved to GitHub Issue: [#1442](https://github.com/OAI/OpenAPI-Specification/issues/1442) and the 'overlay' feature proposal to be included in a future release of the OpenAPI Spec.
