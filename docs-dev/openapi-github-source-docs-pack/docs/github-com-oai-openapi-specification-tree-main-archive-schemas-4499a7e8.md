# Archive of outdated JSON Schema Files

[](https://github.com/OAI/OpenAPI-Specification/tree/main/_archive_/schemas#archive-of-outdated-json-schema-files)

Tip

JSON Schema files for validating OpenAPI descriptions using current OpenAPI versions are available on [https://spec.openapis.org/#openapi-specification-schemas](https://spec.openapis.org/#openapi-specification-schemas).

These schema files are maintained in the `src/schemas` folder of the corresponding `vX.Y-dev` branch in this repository.

Caution

Schema files in this folder are not maintained any more and are not intended for productive use.
